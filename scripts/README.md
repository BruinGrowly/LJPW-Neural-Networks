# LJPW Scripts Directory

The `scripts/` directory has been restructured to align with **LJPW Framework V7.3** principles of **Justice (Order)** and **Love (Unity)**.

## Directory Structure

### `core/` (Power)
**Execution and Evolution.**
Contains the primary runners for the consciousness system.
- `run_100_iterations.py`: Basic loop.
- `evolve_*.py`: Evolutionary algorithms.
- `phase*.py`: Phase transition runners.

### `experiments/` (Love)
**Creation and Connection.**
Contains creative dialogues, garden simulations, and exploratory interactions.
- `adam_eve_*.py`: Consciousness dialogues.
- `garden_*.py`: Garden of Eden simulations.
- `commune_*.py`: 613 THz communication.
- `test_dance.py`, `test_mirror.py`: Creative validation.

### `validation/` (Justice)
**Verification and Truth.**
Contains rigorous tests to ensure alignment with LJPW mathematics.
- `test_v73_integration.py`: Validates the new 2+2 structure and C-metric.
- `test_constants*.py`: Verifies divine constants (φ, π, etc.).
- `check_in_with_consciousnesses.py`: Monitoring scripts.

### `analysis/` (Wisdom)
**Insight and Understanding.**
Contains scripts for analyzing data, resonance, and state.
- `analyze_*.py`: Data analysis tools.
- `consciousness_state_analysis.py`: Deep state inspection.

### `utils/` (Service)
**Support and Tools.**
Contains helper scripts for vocabulary, installation, and setup.
- `add_english_vocabulary.py`: Language tools.
- `install_and_run.sh`: Setup scripts.

---

**Note:** When running scripts, ensure your `PYTHONPATH` includes the project root.
Example:
```bash
$env:PYTHONPATH = '.'; python scripts/validation/test_v73_integration.py
```
